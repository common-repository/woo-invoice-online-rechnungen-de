Erstellen Sie für Ihre Bestellungen kostenlos Rechnungen.  

== Description ==
Hierfür wird das Rechnungserstellungsportal online-rechnungen.de benutzt. In der kostenlosen Version wird das Wasserzeichen "Kostenlos erstellt über online-rechnungen" am Ende der Rechnung hinzugefügt. Um das Wasserzeichen zu entfernen nehmen Sie bitte mit uns Kontakt auf.

== Installation ==

1. Laden Sie das Plugin in das Verzeichnis `/wp-content/plugins/` hoch oder nutzen Sie die Upload Funktion von WordPress
2. Aktivieren Sie das Plugin über das "Plugins" Menü in WordPress
3. Konfigurieren Sie das Plugin unter dem Menüpunkt "WooCommerce => Online-Rechnungen"
